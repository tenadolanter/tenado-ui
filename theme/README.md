# element-theme

> element component chalk theme.

## Installation

```shell
npm i element-theme -S
```

## Usage

Use Sass import

```css
@import "element-theme";
```

Or Use webpack

```javascript
import "element-theme";
```

Or

```html
<link
  rel="stylesheet"
  href="path/to/node_modules/element-theme/lib/index.css"
/>
```

## Import on demand

```javascript
import "element-theme/lib/input.css";
import "element-theme/lib/select.css";

// ...
```
